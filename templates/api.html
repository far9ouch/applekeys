API Documentation

Integrate AppleKEY into your website or application using our simple API.

GET /get-key

Generate a new Apple TV+ key.

// Using fetch
fetch('https://applekeys.onrender.com/get-key')
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log('Your key:', data.key);
    }
  });

Response:

{
  "success": true,
  "key": "XXXXXXXXXXXX"
}

GET /queue-status

Check the number of available keys in the queue.

// Using fetch
fetch('https://applekeys.onrender.com/queue-status')
  .then(response => response.json())
  .then(data => {
    console.log(`Available keys: ${data.keys_available}`);
  });

Response:

{
  "keys_available": 5,
  "max_queue_size": 10
}

Integration Example

Here's a complete example of how to integrate the key generator into your website:

<!-- HTML -->
<button id="getKey">Get Apple TV+ Key</button>
<div id="result"></div>

<script>
// JavaScript
document.getElementById('getKey').addEventListener('click', async () => {
    const result = document.getElementById('result');
    result.textContent = 'Generating key...';
    
    try {
        const response = await fetch('https://applekeys.onrender.com/get-key');
        const data = await response.json();
        
        if (data.success) {
            result.textContent = `Your key: ${data.key}`;
        } else {
            result.textContent = 'Error: Please try again later';
        }
    } catch (error) {
        result.textContent = 'Error connecting to server';
    }
});
</script>
